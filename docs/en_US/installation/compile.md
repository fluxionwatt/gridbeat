# Build from source

## Installation Build Tools

```bash
# wget https://go.dev/dl/go1.25.5.linux-amd64.tar.gz
# tar xvzf go1.25.5.linux-amd64.tar.gz -C /usr/local
```

## Compilation

```
$ git clone https://github.com/fluxionwatt/gridbeat.git
$ cd gridbeat
$ task build
```

:::tip
:::

## Operation

```
$ cd build
$ ./gridbeat server
```
