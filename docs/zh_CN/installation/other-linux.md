# 使用 .tar.gz 包安装

## 下载

根据不同版本及架构下载安装包，例如：

```bash
$ wget https://github.com/fluxionwatt/gridbeat/releases/download/v1.0.0/gridbeat-linux-v1.0.0-amd64.tar.gz
```

## 安装

```bash
$ tar -zxvf gridbeat-linux-v1.0.0-amd64.tar.gz
$ cd gridbeat-linux-v1.0.0-amd64
```

## 启动

执行如下指令启动 GridBeat：

```bash
$ ./gridbeat -h
```
