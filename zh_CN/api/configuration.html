配置 ​

GridBeat 将为 IIoT 平台提供一系列 API 服务，用于查询基本信息、控制网关行为或设置轮询配置。 IIoT 平台必须通过向 GridBeat 发送请求消息来启动通信。 通过返回，GridBeat 将返回所需的信息或执行相应的操作。 如果有错误，将返回一个错误代码来说明失败的原因。

    // node name

    "level": "debug",

    "core": true
}

core 字段选填，默认为 true。
